- [Digraphs](https://digraphs.github.io/Digraphs/) deposited GAP package;
  <https://doi.org/10.5281/zenodo.17211546>
- [Semigroups](https://semigroups.github.io/Semigroups/) deposited GAP
  package; <http://dx.doi.org/10.5281/zenodo.592893>
- [SgpDec](https://gap-packages.github.io/sgpdec/) - Hierarchical composition
  and decomposition of finite permutation groups and transformation semigroups
  (by A. Egri-Nagy and C. Nehaniv with some contributions by me) deposited GAP
  package.
- (with A. Distler) [Smallsemi](https://gap-packages.github.io/smallsemi/) - a library of small
  semigroups deposited GAP package.
- (with R. Cirpons and J. Edwards) [libsemigroups][] C++ library for semigroups and monoids;
  <https://doi.org/10.5281/zenodo.1437752>
- [HPCombi][] Fast combinatorics in C++ using SSE/AVX instruction sets,
  (by F. Hivert with some contributions by me).
- (with R. Cirpons, J. Edwards, M. Tsalakou, and M. Whyte)
  [libsemigroups_pybind11](https://libsemigroups.github.io/libsemigroups_pybind11/)
  python package for using [libsemigroups][] via
  [pybind11](https://pybind11.readthedocs.io/en/stable/);
  <https://doi.org/10.5281/zenodo.17234506>.
- (with [M. Tsalakou][])[step_hen](https://step-hen.readthedocs.io/en/latest/) a rudimentary python
  implementation of Stephen's procedure for inverse semigroups, 2021.
- [todd_coxeter.py](https://gist.github.com/james-d-mitchell/6b06bc78e2bcdb6dfef53a2654d9f953)
  reference implementation of the two main variants of the Todd-Coxeter
  algorithm for semigroups and monoids, 2021.
