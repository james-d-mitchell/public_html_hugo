## GAP

- [Digraphs](https://digraphs.github.io/Digraphs/) deposited GAP package, 2021.
- [Semigroups](https://semigroups.github.io/Semigroups/) deposited GAP
  package, 2021.
- [SgpDec](https://gap-packages.github.io/sgpdec/) - Hierarchical composition
  and decomposition of finite permutation groups and transformation semigroups
  (by A. Egri-Nagy and C. Nehaniv with some contributions by me) deposited GAP
  package, 2019.
- [Smallsemi](https://gap-packages.github.io/smallsemi/) - a library of small
  semigroups (with A. Distler) deposited GAP package, 2019.

## C++

- [libsemigroups][] C++ library for semigroups and monoids, 2021.

## python

- [libsemigroups\_pybind11](https://libsemigroups.github.io/libsemigroups_pybind11/)
  python package for using [libsemigroups][] via [pybind11](https://pybind11.readthedocs.io/en/stable/), 
  (with [M.  Tsalakou][]), 2021.
- [step\_hen](https://step-hen.readthedocs.io/en/latest/) a rudimentary python
  implementation of Stephen's procedure for inverse semigroups (with [M.
  Tsalakou][]), 2021
- [todd\_coxeter.py](https://gist.github.com/james-d-mitchell/6b06bc78e2bcdb6dfef53a2654d9f953) 
  reference implementation of the two main variants of the Todd-Coxeter
  algorithm for semigroups and monoids, 2021

[M. Tsalakou]: https://mariatsalakou.github.io/
[libsemigroups]: https://libsemigroups.readthedocs.io/en/latest/
